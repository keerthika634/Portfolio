PROJECTS


IOT

  • Project Name: Automatic IV Bag Monitoring System Using IoT Data Transmission

The "Automatic IV Bag Monitoring System Using IoT Data Transmission" project automates the monitoring of glucose levels in IV bottles to improve hospital efficiency. Using Arduino IDE 1.8.12, the system utilizes a load cell and amplifier to measure bottle weight and calculate liquid levels. It sends SMS alerts via a GSM modem when the bottle is half full or empty, reducing manual checks. An LED light also indicates the IV bag's status to patients and caregivers, helping prevent the serious risk of air bubbles entering the bloodstream.
